# Joshua Hohaia - Personal Portfolio

A retro-themed personal portfolio website designed to showcase my experience, skills, and projects as a Frontend Developer. The site features a nostalgic CRT monitor aesthetic, interactive elements, and a responsive design.

## üåü Features

*   **Retro CRT Aesthetic:** Custom CRT scanline and text-shadow effects with a user-adjustable intensity slider.
*   **Theme System:** toggle between Light and Dark modes (system preference detected automatically).
*   **Interactive Game:** A built-in "Jumping" game where you collect letters to spell my name. Includes character selection (including GitHub avatar integration).
*   **Responsive ASCII Art:** ASCII banners that adapt to different screen sizes.
*   **Experience Timeline:** An expandable, data-driven timeline showcasing my professional history.
*   **No Frameworks:** Built entirely with vanilla HTML, CSS, and JavaScript for maximum performance and control.
